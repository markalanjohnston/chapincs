Minecraft MakeCode Lessons

Code the Agent. Build the world. Learn programming fundamentals — in any CS class.

About these lessons

Each Minecraft lesson is self-paced. Open the lesson page, follow the steps, and work at your own speed. Every lesson ends with a small digital deliverable — usually a screenshot of your code and a screenshot of what you built in Minecraft.

The lessons cover programming concepts every introductory CS course teaches — sequence, events, debugging, loops, conditionals, and procedures — with the Minecraft Agent as your test subject. They're written to be class-agnostic: usable in Fundamentals of CS, AP CSP, AP CSA, a coding club, or a one-off Hour of Code.
