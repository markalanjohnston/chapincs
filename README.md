# chapincs

Static site for the Chapin High School Computer Science program, served at [chapincs.com](https://chapincs.com) via GitHub Pages.

## Structure

- `index.html` — landing page
- `display.html` — display/kiosk view
- `images/` — recruitment posters and character art
- `CNAME` — GitHub Pages custom domain

## Local preview

Open `index.html` directly in a browser, or run a simple local server:

```sh
python3 -m http.server 8000
```

Then visit http://localhost:8000.
